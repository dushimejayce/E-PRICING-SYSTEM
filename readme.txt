Installation steps:

	1) clone this repo => git clone https://github.com/mucyomiller/market.git
	2)create .env file using .env.example file and add database credintials
	3)make composer install to install and updates required packages
	4)generate app key with=>php artisan key:generate
  now you can watch it offline with local server using:
	php artisan serve
	see it at => http://localhost:8000/
